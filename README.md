# ⭐ Cemi学英语 - 儿童幼升小英语学习App

面向5-6岁儿童的交互式英语学习网页应用，帮助从幼儿园到小学顺利过渡。

## 功能模块

| 模块 | 说明 |
|------|------|
| 🔤 **字母学习** | A-Z 26个字母，点击卡片学发音和单词，支持朗读全部和自动播放 |
| 📚 **单词认知** | 入门、动物、颜色、数字、水果、身体 6大分类，点击发音 |
| 🎤 **大声朗读** | 看→听→读 三步互动，语音识别检测跟读 |
| 🧠 **趣味测验** | 10题一轮，看图选词/中英配对，题目自动朗读 |
| 🃏 **配对游戏** | 翻翻乐配对，英文单词配图片，计时挑战 |
| 🏆 **我的成就** | 星星和进度追踪 |

## 特色

- 🐼 **熊猫盼盼** — 吉祥物陪伴学习，鼓励互动
- 🔊 **音效反馈** — 答对/答错/胜利各有不同音效
- 🎊 **庆祝动画** — 星星爆炸、彩色纸屑、盼盼跳舞
- 📱 **PWA支持** — 可添加到主屏幕，离线可用

## 在线访问

👉 https://cemilearnenglish.github.io/english-app/

## 平板使用

1. 在平板浏览器打开上面的链接
2. 点击浏览器菜单 → "添加到主屏幕"
3. 之后可离线使用，像原生App一样全屏运行

> 兼容 iPad iOS 12.5+ 及 Android 平板。

## 本地使用

直接用浏览器打开 `index.html` 即可，无需安装任何依赖。
